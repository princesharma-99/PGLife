# 🏠 PGLife – Full Stack Web Development Project

PGLife is a full-stack web application developed as part of the **Internshala Full Stack Web Development with AI Training**. The platform helps users search for Paying Guest (PG) accommodations, explore property details, create accounts, log in securely, and save their favorite properties.

---

## ✨ Features

- 🔐 User Registration & Login
- 🏙️ Browse PGs by City
- 🏠 Property Listing & Details
- ❤️ Save Interested Properties
- ⭐ Property Ratings & Amenities
- 📱 Responsive Design using Bootstrap
- 🔒 PHP & MySQL Authentication System

## ⚙️ Installation

1. Clone the repository

```bash
git clone https://github.com/princesharma-99/PGLife.git
```

2. Move the project to your XAMPP `htdocs` folder.

3. Start **Apache** and **MySQL**.

4. Create a database named:

```text
pglife
```

5. Import the SQL file into MySQL.

6. Open your browser and visit:

```text
http://localhost/PGLife
```

---

## 🗄️ Database Configuration

Update the database credentials in:

```text
includes/database_connect.php
```

Default configuration:

- Host: 127.0.0.1
- User: root
- Password: (empty)
- Database: pglife
- Port: 3306
